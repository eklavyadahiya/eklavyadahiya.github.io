Blog Posts Are In The Works

  • January 23, 2021January 23, 2021

Everybody and their mothers is blogging now. I finally got around to setting up this site. I plan to write a post about how I did it so others can do it too. And while some tech pundits might argue that the way I did this was very ‘inefficient’ – my argument to them would be that if you understand how to use certain tools and achieve the desired outcome, how efficient your process is depends completely on your ability to work with those tools.

How I set up this site for free:

  1. Install wordpress on your local computer
  2. Use the Simply Static plugin to generate a static version of the site
  3. Push to github, make sure your repo is setup properly to show up on github pages
  4. Point your domain to github pages, do the necessary linking

Though, one could argue that it isn’t really ‘free’. I still pay for the domain name. And i’ve spent plenty of time learning to do all these things myself. So, there’s definitely some time that has been invested.
